Schnellcheck-Auswertung

Ergebnis-Codes einlesen, je Aufgabe auswerten, lokal gespeichert
nur im Browser
Alle Daten bleiben rein lokal in diesem Browser (kein Upload, kein Server). Auto-Speichern in eine eigene Datei geht nur in Chrome oder Edge. Es bleibt ein unbeaufsichtigter Selbsttest: ein Signal für das Lerngespräch, kein sicherer Leistungsnachweis.
Klasse Erhebung
Ergebnis-Codes einfügen (ein Code pro Zeile, umliegender Text egal)
werden zu einer neuen Erhebung in der gewählten Klasse
